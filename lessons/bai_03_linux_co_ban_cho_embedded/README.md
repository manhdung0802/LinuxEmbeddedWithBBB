# Khung Thực Hành

## Mục đích
- `examples/`: Code mẫu minh họa cho bài học.
- `exercises/`: Bài tập để tự làm.
- `solutions/`: Đáp án tham khảo cho bài tập.

## Gợi ý tổ chức file
- Mỗi bài tập nên có tiền tố số thứ tự, ví dụ: `ex01_...`, `ex02_...`.
- Nếu có nhiều file C, tạo thêm thư mục con theo chủ đề.
- Thêm `README.md` ngắn trong từng thư mục con để ghi cách build/chạy.

## Mẫu lệnh build C nhanh
```bash
gcc -o app main.c
./app
```
