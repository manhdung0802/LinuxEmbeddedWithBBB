# Bài 6: Memory-Mapped I/O trong kernel

## Mục tiêu
- ioremap() / iounmap() từ kernel space
- readl() / writel() để truy cập thanh ghi
- platform_get_resource() để lấy physical address từ DT
- So sánh MMIO trong kernel vs userspace (/dev/mem)
